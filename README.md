# Smart Queue Management System

## Project Overview
The Smart Queue Management System is a web-based platform developed to reduce physical waiting lines and improve queue management in hospitals, banks, billing centers, and service offices.

The system allows customers to generate digital tokens online and monitor live queue status in real time.

---

## Features
- Digital Token Generation
- Real-Time Queue Tracking
- Estimated Waiting Time
- Admin Dashboard
- Multiple Service Queues
- Firebase Real-Time Updates
